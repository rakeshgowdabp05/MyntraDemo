# MyntraDemo

MyntraDemo is a Java MVC ecommerce web application inspired by Myntra-style shopping flow.

## Tech Stack
- Java 21
- JSP and Servlets
- JDBC
- MySQL
- Maven
- Apache Tomcat 10.1
- HTML, CSS, JavaScript

## Features
- Home page
- Product listing and filtering
- Product details
- Wishlist
- Cart
- Checkout
- Orders
- Order details
- Help/contact pages
- Profile dropdown
- Account pages
